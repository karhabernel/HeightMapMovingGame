DX를 이용하여 제작한 지형에 따라 움직이는 큐브 게임입니다.
ImGui 조작을 통해 큐브 텍스쳐, 지형, 지면 텍스쳐(타일, 높이에 따른 지형변화 포함), 하늘 텍스쳐 및 광원 방향의 조정이 가능합니다.

GameTest.zip 내 GaemTest 바로가기를 통해 포트폴리오 내용 확인이 가능합니다.
